# Guides Content Guide

Purpose
- How-tos, build guides, walkthroughs, meta tips, loadouts.

Front matter template
```md
---
title: "Actionable guide title"
author: "Your Name"
date: "2026-03-10"
category: "guides"
platform: "PC"          # or platform list
image: "https://..."
excerpt: "What this guide covers."  # optional
type: "original"        # usually original; curated allowed
source: "Source Site"   # required if curated
sourceUrl: "https://..."# required if curated
---
```

Rules
- Filename = slug (lowercase-hyphen). Example: `elden-ring-faith-build.md`.
- Use ordered/unordered lists for steps; headings for sections.
- ISO dates for ordering; avoid raw HTML/scripts.
- Curated guides must cite source + link.
