# Q&A Content Guide

Purpose
- Short answers to common player questions; FAQs and primers.

Front matter template
```md
---
title: "Question phrased clearly?"
author: "Your Name"
date: "2026-03-10"
category: "qa"
platform: "Multi-platform"   # or specific platform
image: "https://..."
excerpt: "One-line answer/summary." # optional
type: "original"        # or "curated"
source: "Source Site"   # required if curated
sourceUrl: "https://..."# required if curated
---
```

Rules
- Filename = slug (lowercase-hyphen). Example: `can-you-pause-elden-ring.md`.
- Keep answers concise; use headings for variants/steps.
- ISO dates for ordering; no raw HTML/scripts.
- Curated answers must include source + link.
